# Catalog Cross-Reference

Future catalog repository: `github.com/pay-bye/agent-os-catalog`

The catalog is discovery only. It helps adopters find the Codex app-server driver, supported
versions, manifests, and release material. It does not decide compatibility.

Catalog entries for the Codex app-server driver point to:

- repository: `github.com/pay-bye/agent-os-driver-codex-app-server`
- driver manifest: `internal/install/driver_manifest.json`
- release metadata from `github.com/pay-bye/agent-os-driver-codex-app-server/releases`
- invocation contract from `github.com/pay-bye/agent-os`

Adopters verify compatibility from those component-owned sources.
