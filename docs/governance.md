# Governance

The root `GOVERNANCE.md` file defines maintainer authority for the Codex app-server driver.

Compatibility ownership stays with component-owned sources:

- published invocation contract from `github.com/pay-bye/agent-os`
- driver manifest: `internal/install/driver_manifest.json`
- release metadata from `github.com/pay-bye/agent-os-driver-codex-app-server`

The future `github.com/pay-bye/agent-os-catalog` repository is discovery only. It may link to this
material, but it does not supersede or override component-owned truth.
