# Security

Do not report vulnerabilities through public issues.

## Reporting

The public vulnerability reporting surface is GitHub private vulnerability reporting for
`github.com/pay-bye/agent-os-driver-codex-app-server` after the public repository exists. Before U6
creates public release surfaces, there is no public security intake for adopters.

Reports include:

- affected version or commit
- reproduction steps
- observed impact
- logs with secrets and payloads removed

## Supported Versions

Supported public versions are listed in release metadata once U6 publishes release artifacts. Until
then, no public Codex app-server driver version is supported for adopter production use.

## Disclosure

Maintainers coordinate fixes through private advisories, release patched artifacts, and publish
advisory details after remediation is available.
